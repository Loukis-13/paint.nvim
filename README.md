# Paint.nvim
Painting tool plugin for neovim inspired in MS Paint and Durdraw

## Features
- [x] Keyboard drawing
- [x] Mouse drawing
- [x] Save/Load
- [ ] Undo/Redo
- [ ] Shapes
  - [ ] Line
  - [ ] Square
  - [ ] Circle
  - [ ] Triangle
- [ ] Tools
  - [x] Pencil
  - [x] Eraser
  - [x] Eye drop
  - [x] Fill
  - [ ] Text
  - [ ] Color picker
- [x] User command arguments
- [x] Choose pencil type / Char list
